## 📖 Overview
This tool acts as a hardened gateway that sits between your AI agents and model providers to neutralize prompt injection attacks. Developers deploying self-hosted AI stacks often expose sensitive data because they treat LLMs as trusted components. This firewall operates as a transparent proxy that validates and sanitizes requests before they reach the inference engine. It also inspects responses to ensure they are safe before returning them to the agent. It includes features like PII redaction to protect personally identifiable information during processing.
